this directory stores the following files: 
- phonebook.xml        --- Fritz Box phone book created by egw2fbox.pl, uploaded by fritzuploader.pl
- phonebook.hash       --- helper file to avoid unnecessary Fritz Box updates, created by cronjob.sh
- egw_timestamps.data  --- helper file to avoid unnecessary DB access, created by egw2fbox.pl
- muttrc_addresses     --- the address book to be used with MUTT created by egw2fbox.pl, see etc/muttrc.example
